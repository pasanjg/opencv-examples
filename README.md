### Install Packages

- **OpenCV**

```
$ pip install opencv-python
```

- **Matplotlib**

```
$ pip install matplotlib
```

- **Numpy**

```
$ pip install numpy
```



### Usage

```
$ git clone https://github.com/pasanjg/opencv-examples.git
$ cd opencv-examples
$ cd <directory>
$ python <opencv_python_filename>.py
```
