### Setup project

```
$ git clone --depth 1 --single-branch https://github.com/pasanjg/opencv-examples.git
```


### Install Packages
Inside the project root,

```
$ pip install -r requirements.txt
```



### Usage

```
$ cd <directory>
$ python <opencv_python_filename>.py
```
